الملفات وشرح وظيفة كل واحد
🚀 main.py Python
القلب الرئيسي للنظام — يحتوي على كل الـ API endpoints ويربط كل المكونات ببعض
  • تعريف التطبيق FastAPI مع CORS middleware
  • endpoint رفع الـ CV: POST /candidates/upload
  • endpoint البحث الفوري: POST /find-jobs (يرفع CV ويرجع الوظائف المناسبة فوراً)
  • endpoints إدارة الوظائف والمرشحين والمطابقة
  • Admin endpoints: verify password / seed jobs
  • حل مشكلة تكرار المهارات (duplicate skills fix)
  • تصدير المطابقات كـ CSV
🗄️ database.py Python
تعريف جداول قاعدة البيانات باستخدام SQLAlchemy ORM والعلاقات بينها
  • جدول candidates: اسم، إيميل، تيليفون، موقع، خبرة، تعليم، نص الـ CV
  • جدول skills: مخزن موحد لكل المهارات
  • جدول job_descriptions: الوظائف مع الحد الأدنى من الخبرة والتعليم
  • جدول وسيط candidate_skills: many-to-many بين المرشحين والمهارات
  • جدول وسيط job_skills: many-to-many بين الوظائف والمهارات
  • دالة init_db(): تنشئ الجداول أول مرة وتعمل safe migration للـ columns الجديدة
🔍 cv_parser.py Python
استخراج النص من ملفات PDF/DOCX ثم تحليله لاستخراج البيانات المنظمة
  • استخراج النص من PDF باستخدام pdfplumber
  • استخراج النص من DOCX باستخدام python-docx
  • قائمة KNOWN_SKILLS تحتوي على 160+ مهارة (تقنية، مالية، إدارية، إعلامية، هندسية…)
  • دالة _extract_email(): regex لاستخراج الإيميل
  • دالة _extract_experience(): تحسب سنوات الخبرة من التواريخ
  • دالة _extract_education(): يحدد أعلى مستوى تعليمي مذكور
  • دالة parse_cv(): تجمع كل شيء — تجرب Groq AI أولاً، لو فشل ترجع لـ regex
🤖 ai_service.py Python
التكامل مع Groq AI (LLaMA 3.1 8B) لتحليل CVs وشرح نتائج المطابقة
  • دالة parse_cv_claude(): يرسل نص الـ CV لـ Groq ويطلب JSON منظم (اسم، إيميل، مهارات، خبرة، تعليم)
  • البرومبت محسّن خصيصاً لـ CVs منطقة الخليج العربي (تقنية + غير تقنية)
  • دالة explain_match(): يكتب جملتين يشرح فيهم ليه المرشح مناسب للوظيفة
  • دالة summarise_profile(): يكتب ملخص تشجيعي للمرشح بعد رفع الـ CV
  • دالة _fallback_explanation(): نص جاهز لو فشل الـ AI
⚖️ matcher.py Python
منطق حساب نسبة التطابق بين المرشحين والوظائف
  • دالة match_candidates(job, db): تجيب كل المرشحين وتحسب score لكل واحد لوظيفة معينة
  • دالة recommend_jobs(candidate, db): تجيب كل الوظائف وتحسب score لمرشح معين
  • حساب skill_score: نسبة المهارات المتطابقة (55%)
  • حساب experience_score: مقارنة سنوات الخبرة (30%)
  • حساب education_score: مقارنة مستوى التعليم (15%)
  • دالة extract_skills_from_text(): تستخرج المهارات من نص الوظيفة
📋 schemas.py Python
نماذج Pydantic لتحديد شكل البيانات المرسلة والمستقبلة في الـ API
  • CandidateOut: بيانات المرشح المرتجعة من الـ API
  • JobCreate: شكل إنشاء وظيفة جديدة
  • JobOut: بيانات الوظيفة المرتجعة
  • MatchResult: نتيجة مطابقة مرشح بوظيفة (مع الـ score والمهارات)
  • JobMatchResult: نتيجة توصية وظيفة لمرشح
  • StatsOut: إحصائيات لوحة التحكم
🌱 seed_jobs.py Python
يحتوي على 26 وظيفة نموذجية متنوعة لاختبار النظام
